# VitePress 系列教程(04)：Markdown 拓展语法

markdown 语法其实已经足够好用了，但是某些时候还是不够灵活。例如 markdown 的语法只有基本的标签，比较难实现在其中布局、写样式；调试
markdown 中的代码，需要保持文件与代码同步；markdown 文件不支持相互引用嵌入。这些问题在 Markdown Extensions 中都可以解决。
